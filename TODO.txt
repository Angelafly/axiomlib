== TaggedDataSet == 

Набор траекторий, каждой из которых соответствует множество строковых тегов ("reference", "testing", "1", "normal", etc.)
Основные операции:
- Взатие упорядоченного набора траекторий по множеству тегов tagset: возвращает траектории t, для каждой из которых множество тегов, ассоциированное с t, содержит множество тегов tagset.
- Добавление траектории, с которой ассоциировано множество тегов

Нужно реализовать построение такого множества по стандартным наборам данных, а также разработать формат конфиг-файла, специфицирующий произвольный набор данных такого вида.
